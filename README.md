Developed a comprehensive Sports Analytics Engine processing 350 matches and 11K+ runs to visualize the career evolution and strike-rate consistency of MS Dhoni.

Engineered an "Innings Builder" metric using stacked bar charts to analyze Strike Rotation % vs. Boundary Dependency %, quantifying batting efficiency against world-class bowling attacks.

Implemented multi-dimensional batting analysis (Runs by Position, Ground, and Opposition), using DAX to isolate performance spikes in high-pressure scenarios.

Designed a custom Strike Rate Donut breakdown and interactive opposition filters to audit performance variations across 15+ years of international cricket data.
